# Freelancing Platform

A full-stack freelancing platform that connects Clients and Freelancers through project bidding and project management workflows. The application includes role-based access for Admin, Client, and Freelancer users.

## 🚀 Features

### Client

* Register and Login
* Create Projects
* View Own Projects
* View Freelancer Bids
* Accept or Reject Bids
* Track Project Status

### Freelancer

* Register and Login
* Browse Available Projects
* Place Bids on Projects
* View My Bids
* Complete Assigned Projects

### Admin

* Manage Users
* Manage Projects
* Manage Bids
* Monitor Platform Activities

## ⚙️ Installation & Setup

### Backend

1. Open Backend project in Eclipse/STS
2. Configure MySQL Database
3. Update application.properties
4. Run Spring Boot Application

### Frontend

1. Open Frontend folder in VS Code

2. Install dependencies

```bash
npm install
```

3. Start application

```bash
npm start
```

Frontend URL:

```text
http://localhost:3000
```

Backend URL:

```text
http://localhost:8080
```
